# VerbaFlow - AI 语流 · 术语管理与校对套件

> **[English Documentation](./README.md)**

VerbaFlow 是一款专为音视频内容创作者、译者和校对人员设计的专业级生产力工具。它利用 Google Gemini 模型，在确保内容完整性的前提下，提供强大的术语管理、自动化校对和术语一致性检查功能。

## 核心功能

### 1. 🎧 校对工作台 (Studio) & 智能分析
*   **多格式支持**：支持上传音频 (MP3, WAV) 或 视频 (MP4, MOV)。
*   **深度分析**：自动提取专有名词、技术术语，并检查上下文一致性。
*   **摘要生成**：自动生成内容摘要、发言人列表和会议议程。
*   **视频预览**：内置视频播放器，支持字幕同步滚动，点击字幕即可跳转至对应视频画面。

### 2. 📚 术语知识库 (Knowledge Base)
*   **语境提取**：AI 分析文稿，基于特定上下文生成术语定义。
*   **智能导出**：从当前工作会话中自动提取专业术语表。
*   **闭环复核**：将整理好的术语库反哺给 AI，对文稿进行二次复核，确保术语严格一致。
*   **Drive 集成**：支持将术语库同步保存至 Google Drive（模拟集成）。

### 3. ✍️ 人工介入复核 (Human-in-the-loop)
*   **交互式表格**：左侧原文、右侧修正，并列对比，高效复核。
*   **精准定位**：点击任意术语即可播放对应的音视频片段，辅助判断。
*   **自定义状态**：标记条目为“已确认”、“需复核”或添加自定义标签。
*   **AI 指令**：支持输入额外指令（如“演讲者有口音”、“保留口语化表达”）以获得更精准的 AI 分析。

### 4. 📤 双模式输出
*   **精校 SRT**：生成修正后的字幕文件，**严格保持原始时间轴不变**。
*   **Markdown 文稿**：生成可读性强的文章式文稿，适合发布博客或文档。

### 5. 🤖 AI 智能助手
*   **上下文感知**：悬浮式 AI 助手，了解当前的校对任务背景。
*   **会话管理**：支持会话自动命名、历史记录管理以及人工重命名。
*   **模型切换**：可随时在 Gemini Flash (速度优先) 和 Gemini Pro (推理优先) 之间切换。

## 技术栈

*   **前端**: React 19, TypeScript, Tailwind CSS
*   **AI 引擎**: Google GenAI SDK (Gemini 1.5 / 2.0 / 3.0 模型)
*   **图标库**: Lucide React
*   **工具**: 自研 SRT 解析器与 Markdown 渲染

## 快速开始

1.  **克隆仓库**
    ```bash
    git clone https://github.com/yourusername/verbaflow.git
    cd verbaflow
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **配置 API Key**
    在环境变量中设置您的 Google Gemini API Key。
    ```bash
    export API_KEY="your_google_api_key_here"
    ```

4.  **启动开发服务器**
    ```bash
    npm start
    ```
